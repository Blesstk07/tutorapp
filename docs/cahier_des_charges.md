# Cahier des charges - TutorApp

## Titre du projet

Conception et développement d'une plateforme web de mise en relation entre étudiants et tuteurs universitaires.

## Contexte

De nombreux étudiants rencontrent des difficultés dans certaines matières universitaires et recherchent un accompagnement personnalisé. Les tuteurs disposent des compétences nécessaires mais manquent souvent de visibilité.

## Objectif général

Développer une plateforme web permettant aux étudiants de rechercher, contacter et réserver des séances avec des tuteurs universitaires.

## Acteurs

### Étudiant

* Créer un compte
* Se connecter
* Rechercher un tuteur
* Consulter les profils
* Réserver une séance
* Évaluer un tuteur

### Tuteur

* Créer un compte
* Gérer son profil
* Consulter les réservations
* Accepter ou refuser une demande

### Administrateur

* Gérer les utilisateurs
* Gérer les tuteurs
* Consulter les statistiques

## Technologies

### Frontend

* HTML5
* CSS3
* JavaScript

### Backend

* Java Spring Boot

### Base de données

* MySQL

## Résultat attendu

Une application web permettant la gestion complète de la relation entre étudiants et tuteurs universitaires.
